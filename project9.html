THIS WEBSITE

This website started as a project for a interactive design class, but in the processing of making it, I had developed a interest for interaction design and this website became my first interaction project.

This website is built from the scratch with HTML and CSS as the majority of the code.

To start the design, I first I had to determine what color scheme would work well with the contents that I would be putting on it, and I decide to go with the classical dark grey that would compliment a broad range of colors.

Then I moved to the arrangement of the contents: what is the best way to showcase my work with the skill I have? Namely, only HTMl and CSS.

In the end, I created a scheme of using color gradient of the work as background to accentuate and compliment the work itself while immersing the viewers in the work without vying for their focuses.
